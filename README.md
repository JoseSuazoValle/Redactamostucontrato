# Redactamostucontrato
  Es la aplicación que permite añadir nuevos contratos y preparar los formularios para el auto rellenado. Los usuarios que ingresan a la notaria interactúan con un tótem, el cual solicita:  
  
    1.- Seleccionar el tipo de contrato.  
    
    2.- Ingresar la información de manera secuencial la que se auto rellena en el contrato original.  
    

# appMeson
  Es la aplicación que recibe los contratos rellenados en el tótem y permite:  
  
    1.- Gestionar la veracidad de la información.  
    
    2.- Editar el contrato.  
    
    3.- Imprimir y emitir el cobro del contrato.
